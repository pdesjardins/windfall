# Keyboard Reference

## Unit Queue

| Key | Action |
|---|---|
| `Space` | Skip — remove the selected unit from the queue for this turn |
| `W` | Wait — defer the selected unit to the end of the queue |
| `F` | Encamp / Anchor — put the selected unit to sleep until you wake it |
| `B` | Build — open the build menu for the selected crew unit (only appears when on a buildable hex with AP remaining) |
| `U` | Unload — wake all sleeping crew aboard the selected ship so they can disembark (only appears when ship is adjacent to land) |

Skipped units do not act again until the next turn. Encamped and anchored units persist across turns until you click them to wake them.

## Build Menu

When `B` is pressed with a crew unit selected on a buildable hex, the info panel shows a numbered list of available improvements.

| Key | Action |
|---|---|
| `1` | Build the first listed improvement |
| `2` | Build the second listed improvement (if available) |
| `Esc` or `B` | Cancel — close the build menu without acting |

Building costs 1 crew AP. Only one improvement is allowed per hex.
